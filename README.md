# 📝 Treinando Blog

## 📌 Sobre o Projeto

Projeto de blog simples desenvolvido para praticar estruturação de conteúdo e layout de páginas.

## 🛠️ Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## ⚙️ Funcionalidades

* Página de posts
* Layout organizado
* Navegação básica

## 🎯 Objetivo

Treinar criação de páginas com conteúdo dinâmico e estrutura de blog.

## 📚 Aprendizados

* Organização de conteúdo
* Estrutura de layout
* Separação de responsabilidades

## ▶️ Como Executar

```bash
git clone https://github.com/DeivsonRibeiro/Treinando_blog.git
cd Treinando_blog
```

Abra no navegador.
